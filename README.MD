Spell Number in English
=======================

num_spell_eng - Spell a number in English (Golang)

## Usage

Import package num_spell_eng

```import github.com/taras-tk/num_spell_eng```

Convert number
```go
    str := num_spell_eng.NumberSpell(23) // outputs "twenty-three"
```
